Проще всего с сервером работать из IDEA, вернее это привычнее для меня, а так подойдет любая среда разработки под java. В IDEA нужно сделать import проекта из исходников указав папку. Дальше она сама должна все распознать и импортировать проект gradle. В левой части будет панель Gradle, в ней Tasks -> Other -> generateProto Но можно запустить и вручную: из папки uTransnetCalcServer вызвать команду gradlew generateProto

Аналогично для запуска сервера gradlew bootRun или в IDEA найти файл WebUIStarter -> ПКМ -> Run или Debug.

Вэб интерфейс запуститься на http://localhost:8080/ и http://localhost:8080/preset
Порт для подключения клиента по умолчанию 8463.

Но на самом деле сейчас нет необходимости в запуске сервера для работы клиента, при старте по таймауту отключиться
